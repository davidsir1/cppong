# Cppong

A simple Pong game implemented in C++ using the SFML library.

## Controls

- **Player 1**
  - Move up: `up arrow`
  - Move down: `down arrow`
- **Player 2**
  - Move up: `W`
  - Move down: `S`

## Requirements

- SFML 2.5
- C++ Compiler (e.g, g++)

## Installation

```bash
git clone https://github.com/davidsir1/cppong
cd cppong
make
```
